# ⚙️ Prerequisites

Before running this project, make sure you have the following installed:
- Node.js (v18 or later recommended)
- npm (comes with Node.js)

You can check if they are installed by running these commands in the command prompt:
node -v
npm -v


# 📦 Installation (Setup)

Clone the repository (again those are commands fot the terminal):
git clone https://github.com/tetrapodus/todo-app.git

Navigate into the project folder:
cd todo-app

Install dependencies:
npm install

Start the development server:
npm run dev

After running the command, open your browser and go to:
http://localhost:5173
